Installation du MKR WiFi 1010

Afin d’utiliser le MKR WiFi 1010 il faut effectuer quelques étapes au préalable :

Ouvrir Arduino IDE.

  • Aller dans l’onglet “Outils”, puis “Type de carte” et cliquer sur “Gestionnaire de carte”. Dans la fenêtre qui s’ouvre rechercher “MKR WiFi 1010” puis installer le paquet qui s’affiche.

  • Aller dans l’onglet “Outils” et cliquer sur “Gérer les bibliothèques”. Ensuite dans le fenêtre que s’ouvre rechercher “WiFiNINA” et installer la bibliothèque.

  • Afin de vérifier les différents paquets on bien été installé, brancher votre MKR WiFi 1010 à l’ordinateur puis aller dans “Fichier”, puis “Examples”, puis “WiFiNINA” et cliquer sur “ScanNetworks”. Ensuite transférer le code sur votre module puis ouvrir le “Moniteur Série”. Le module cherche et affiche sur le moniteur les différents réseau à proximité. Si rien le module ne détecte aucun réseau vérifier l’installation et si les bon paquets sont installés.
