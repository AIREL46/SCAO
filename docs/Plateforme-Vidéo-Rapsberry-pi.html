Plateforme Vidéo Rapsberry pi

Objet

L’objet de la plateforme Vidéo Rapsberry pi est la réalisation des fonctions vidéos à partir d’une carte Rapsberry équipé d’un shield TV. ## Description La réalisation de cette plateforme fait appel à : * pour le matériel, d’une carte Raspberry Pi3 model B+ complétée d’une carte micro SD et d’un tuner TV DVB TV microHAT monté en shield sur le connecteur GPIO * pour les OS, RAPSBIAN et LIBRELEC * pour les applications, TVHEADEND et Kodi. L’application TVHEADEND est installée dans l’environnement RASBIAN ## Tutoriel “Premiere utilisation Raspberry Pi” ## how-to-stream-digital-tv-with-the-raspberry-pi-tv-hat * New Out Of Box Software (NOOBS) is an easy operating system installation manager for the Raspberry Pi NOOBS * Il est installé sur la carte micro-SD, 2 fichiers pose problème, recovery.img et recovery7.img, ils sont normalement exécutable, mais après les avoir copier ils ne sont plus et impossible de changer de statut avec la cde chmod. ## Carte SD * Avec la commande lspci : 04:00.0 Unassigned class [ff00]: Realtek Semiconductor Co., Ltd. RTL8411B PCI Express Card Reader (rev 01) * Avec carte SD “grand format” : impossible de l’identifier * Avec carte micro-SD : miracle - l’identification est OK * N.B. : la réponse à la commande lspci est la même dans les 2 cas (Unassigned class [ff00]).

Molotov :

En référence à l’article de Planète Linux N°102, Molotov permet de regarder toutes les chaînes de la TNT sur un PC. Sous Linux, il n’y a pas d’installation, c’est un fichier AppImage qui fonctionne avec libgconf.
