Réparation de l’ordinateur ASUS

Durant la réparation les disques opérationnels 150MO et 500MO sont installés sur l’ordinateur COMPAQ.

Deux disques de 500MO équipent l’ordinateur ASUS, les OS intallés sont : * Ubuntu sur le premier * Windows XP sur le deuxième (voir ci-dessous).

Lors du démarrage de l’ordinateur, pour choisir entre Ubuntu et Windows XP, presser F8 pour entrer dans le Boot Menu “Select a boot first device , se positionner sur hard disk, valider et choisir le disque adequat.

La configuration de l’ordinateur ASUS Le diagnostic de la panne aléatoire est : plantage lors du déroulement du BIOS. L’écran reste noir. Il y a donc aucun moyen d’investigation, pas même l’entrée dans le menu du BIOS. Cette panne est apparue au fil du temps, parfois le démarrage était possible et parfois c’était le plantage. La proportion entre démarrage et plantage était de plus en plus favorable au plantage.

Installer Windows XP sur un disque dur Sata

installer-windows-xp #### Activer le mode IDE N.B. : Le mode AHCI ne convient pas car il implique de créer un ISO de Windows XP qui contienne les drivers SATA correspondant à la carte mère PTS73. Hors ces drivers sont introuvables sur Internet.

  • Appuyer sur le bouton poussoir de mise en route de l’ordinateur
  • Appuyer sur la touche F2 pour entrer dans le BIOS
  • Se déplacer sur Integrated peripherals et valider
  • Choisir MCP storage configuration [Press enter]
  • Choisir SATA Operation Mode [IDE]. #### Installer Windows XP avec le mode IDE
  • booter sur le CDROM Windows XP et suivre les indications. ### Intallation de la carte graphique
  • Mettre sous tension l’ordinateur
  • entrer dans le BIOS en appuyant sur la touche F2
  • Aller dans Advanced chipset features et valider en appuyant sur la touche Entrée
  • Aller sur Onboard GPU et valider en appuyant sur Entrée
  • deux choix : Enable if no Ext GPU - Always enable
  • cocher Always enable et valider en appuyant sur la touche Entrée
  • sortir du BIOS en appuyant sur F10 (Save).
