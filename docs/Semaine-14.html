Arduino IDE + connection d’une Arduino Uno

Pour l’installation de Arduino IDE
  1. Installation via le terminal: impossible de connecter l’arduino uno;
  2. Installation via Logithèque Ubuntu: impossible de connecter l’arduino uno;
  3. Solution trouvé, suivre les étapes sur le lien donné au dessus.
Connection d’une Arduino Uno:
  1. Ouvrir Arduino IDE;
  2. Trouver le sketch d’exemple “Blink”: FICHIER->EXEMPLES->01.BASICS->Blink;
  3. Dans le fenêtre qui s’ouvre, sélectionnez votre carte (Ici Arduino UNO): OUTILS->TYPE DE CARTE;
  4. Choisissez le port série auquel est branché votre Arduino: OUTILS->PORT SERIE;
  5. Pour mettre le sketch Blink dans votre Arduino, cliquez sur le bouton “TELEVERSER” en à gauche (on peut voir une flèche sur le bouton);
  6. Une fois le téléversement fini vous devriez voir la LED jaune, avec un “L”, clignoter.
