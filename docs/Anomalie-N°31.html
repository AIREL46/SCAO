Fiche d’anomalie N°31 - BIOS - UEFI

Matériel concerné : Ordinateur portable LDLC

Constat

L’ordinateur ne boot pas. ## Analyse L’entrée dans le BIOS setup utility est possible par l’appui sur F2. ## Solution 1) Tout à fait par hasard, dans le BIOS setup utility, à l’onglet boot, j’ai opté pour UEFI Boot Enabled. Sans que je comprenne pourquoi, le choix de cette fonction a permis de débloquer la situation. Ensuite, automatiquement, l’option est redevenue : UEFI Boot Disabled.

  1. Le problème étant réapparu j’ai :
  • retirer le disque HDD
  • installer Ubuntu 20.04 LTS sur le disque SSD
  • formater le disque HDD et le désigner pour la sauvegarde
  • le problème résiduel est que ce disque n’est pas monté automatiquement
  • la sauvegarde automatique n’est pas opérationnelle.
  1. L’ordinateur fonctionne correctement. Le problème était donc uniquement lié au logiciel.
